Snake Game

A simple Snake game implemented in Java.

About the Project

This project is a classic Snake game developed using Java. Players control a snake to eat food, growing longer with each meal while avoiding collisions with walls or itself.

How to Play

  1. Run the Java application.
  2. Use the 'W', 'A', 'S', 'D' keys to control the snake's direction.
  3. Avoid colliding with walls or the snake's own body.
  4. Eat the food to grow longer and increase your score.
