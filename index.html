Snake Game

A classic Snake game implemented in Java.

Overview

This project is a classic Snake game developed using Java. Players control a snake to eat food, growing longer with each meal while avoiding collisions with walls or itself.

Key Features

  • Interactive Java console-based game.
  • Real-time snake movement and collision detection.
  • Dynamic food generation.
  • Score tracking and game over conditions.
